# CardMaker ✨

Personal Quote & Affirmation Card Maker - A beautiful, client-side web application for creating, customizing, and sharing personalized quote and affirmation cards.

## Features

✨ **Create Custom Cards**
- Choose between Quote and Affirmation card types
- Customize text, author/attribution, colors, and font size
- Real-time preview of your card

🎨 **Full Customization**
- Pick any background and text color
- Adjust font size from 14px to 48px
- Add author/attribution or leave it blank

💾 **Persistent Storage**
- All cards are automatically saved to your browser's local storage
- Your cards persist between sessions

📥 **Export Options**
- Download individual cards as high-resolution PNG images
- Export all cards at once as a ZIP file

📱 **Fully Responsive**
- Beautiful design that works on desktop, tablet, and mobile
- Smooth animations and intuitive interface

🚀 **Client-Side Only**
- No server required - runs entirely in your browser
- No data sent to any external server
- Perfect for hosting on GitHub Pages

## Getting Started

1. Clone or fork this repository
2. Enable GitHub Pages in your repository settings
3. Select the `main` branch as the source
4. Visit your GitHub Pages URL to start creating cards!

Or use it locally:
1. Download/clone the repository
2. Open `index.html` in your web browser
3. Start creating your cards!

## Usage

### Creating a Card
1. Select a card type (Quote or Affirmation)
2. Enter your text
3. (Optional) Add an author or attribution
4. Customize the background color and text color
5. Adjust the font size to your liking
6. Click "Create Card" to save it

### Managing Your Cards
- **Preview**: See your card in real-time as you customize it
- **Download**: Hover over a card and click "Download" to save it as a PNG
- **Delete**: Remove individual cards with the "Delete" button
- **Export All**: Download all your cards as a ZIP file
- **Clear All**: Delete all cards at once (be careful!)

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid and Flexbox, animations
- **Vanilla JavaScript**: No frameworks or dependencies
- **html2canvas**: For converting cards to images
- **JSZip**: For creating ZIP exports
- **Local Storage**: For persistent data storage

## Local Storage

This app uses your browser's local storage to save cards. This data:
- Is stored locally on your device
- Is never sent to any server
- Can be cleared at any time through browser settings
- Persists when you close and reopen the app
